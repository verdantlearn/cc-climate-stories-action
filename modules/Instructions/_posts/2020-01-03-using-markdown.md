# Using Markdown

We'll be using a simple language called Markdown to format our collaboratively-created resources

Markdown allows you to add formatting to plain text

```
# Heading 1
## Heading 2
### Heading 3

normal font
**bold**
*italics*

1. Numbered list
2. Second item in list

- Bullet points

- [ ] Task with checkbox
- [ ] Completed task

```
